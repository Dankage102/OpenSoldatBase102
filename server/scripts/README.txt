OpenSoldat Server Scripting README
Updated 2022-07-31

Script Core 3 Notes:
 The Scripting Engine coded into the OpenSoldatServer uses the "Pascal"
 language syntax. You may find full documentation regarding the
 functions and variables/events at the following URL:
 https://wiki.soldat.pl/index.php/Category:ScriptCore_3

 Script Core 3 is implemented using RemObjects PascalScript, which can
 be accessed at https://github.com/remobjects/pascalscript
